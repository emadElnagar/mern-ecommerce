# Mern Ecommerce
ecommerce web app using mern
## Technologies Used
* HTML5
* CSS3
* ReactJS
* Redux
* Material UI
* Material UI icons
* NodeJS
* ExpressJS
* Mongodb
## Features
* Product
  * Review Product
* Cart
  * Add To Cart
  * Remove From Cart
  * Clear Cart
* User
  * Sign Up
  * Log In
  * Update Profile
